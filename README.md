# Overview

This is a static site created for Candid Crypto Podcast, featuring all the podcast episodes, videos, and articles published by us. We aren't making new episodes at the moment, but I still want to preserve all of the stuff our team published over the years.

This repo leverages [Hugo](https://gohugo.io/) and the Hugo theme [*Hugo Blog Awesome*](github.com/hugo-sid/hugo-blog-awesome).

## Quick start

[Install Hugo](https://gohugo.io/installation/) (extended edition), and then run the command `hugo server`.

# Motivation

For years, we ran our site using Wordpress. It served its purpose when we started (quick to set up, built-in post editing, support for comments), but is overkill from a maintenance and cost perspective.
